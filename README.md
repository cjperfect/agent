## 环境要求 
1. node

## 项目启动

### 前端项目启动
1. 进入根目录安装依赖`pnpm install`
2. 启动项目`pnpm dev`

### 工作流启动
1. 本地执行`npx n8n`, 安装n8n并启动项目
2. 安装MCP社区节点
3. 进入n8n_agent文件夹, `main.json`为工作流文件, 导入到n8n当中
4. 激活工作流即可
